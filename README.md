# Platformer Game (Pygame + Tiled)

## Опис проєкту

Це навчальний 2D платформер на **Python** з використанням **Pygame** та мапи, створеної у **Tiled** (`.tmx`).

У грі є:
- гравець з рухом, стрибком та гравітацією;
- анімації персонажа для бігу, стрибка та стану спокою;
- камера/зсув карти по X (скролінг);
- колізії з платформами через object layer `collision`;
- збір пляшок з tile layer `bottles` та їх видалення з мапи;
- підрахунок зібраних пляшок;
- фонова музика.

Проєкт підходить як приклад роботи з:
- **OOP** (класи `Settings`, `Player`, `Map`, `Music`);
- колізіями через `pygame.Rect`;
- системою координат "світ/екран" через зсув карти;
- роботою з ресурсами через універсальний `create_path()`;
- читанням `.tmx` мап за допомогою `pytmx`.

## Встановлення та запуск

### 1) Створити та активувати віртуальне середовище

**Windows (PowerShell):**

```bash
python -m venv venv
venv\Scripts\Activate.ps1
```

**macOS/Linux:**

```bash
python3 -m venv venv
source venv/bin/activate
```

### 2) Встановити залежності

```bash
pip install pygame pytmx
```

За бажанням можна створити `requirements.txt`:

```text
pygame
pytmx
```

### 3) Запустити гру

Точка входу проєкту - `main.py`:

```bash
python main.py
```

Після запуску відкривається вікно гри розміром `1200x800`. FPS обмежено значенням `60`.

## Як працює рівень (Tiled / TMX)

Основна мапа знаходиться у файлі:

```text
assets/tilemap/map.tmx
```

Для коректної роботи логіки у `.tmx` важливі такі шари:

- `collision` - object layer з прямокутниками колізій землі та платформ;
- `bottles` - tile layer з пляшками, які можна збирати.

Під час гри `Map.collision_map()` створює список `pygame.Rect` для колізій. Метод `Map.bottle_remove()` перевіряє зіткнення `Player.HITBOX` з тайлами пляшок, видаляє зібрану пляшку з шару `bottles` та збільшує лічильник `BOTTLE_COUNTER`.

## Основні модулі та логіка

### `main.py`

Точка входу. Імпортує `run()` з пакета `modules` та запускає гру.

### `modules/running.py`

Відповідає за головний цикл:
- обробляє події закриття вікна;
- запускає фонову музику;
- очищає екран;
- рендерить мапу зі зсувом;
- перевіряє колізії;
- оновлює гравця;
- показує лічильник пляшок.

### `modules/app.py`

Містить базові налаштування гри:
- створення вікна `1200x800`;
- назву вікна;
- `FPS = 60`;
- `pygame.time.Clock()`.

### `modules/map.py`

Клас `Map` відповідає за:
- завантаження `.tmx` мапи через `pytmx.load_pygame()`;
- рендер видимих tile layers;
- зсув карти через `MOVE_MAP`;
- створення прямокутників колізій з шару `collision`;
- видалення пляшок з шару `bottles`;
- підрахунок зібраних пляшок.

### `modules/settings.py`

Клас `Settings` використовується як базовий клас для об'єктів із зображенням:
- зберігає розміри та координати;
- завантажує зображення з `assets/media`;
- масштабує та віддзеркалює спрайт;
- рендерить зображення на екран.

### `modules/player.py`

Клас `Player` відповідає за:
- рух вправо/вліво;
- стрибок;
- гравітацію;
- перевірку колізій зверху, знизу, справа та зліва;
- анімації `run1`, `jump`, `breath`;
- напрямок персонажа;
- зсув мапи через `move_map()`.

### `modules/music.py`

Клас `Music`:
- завантажує аудіофайл з `assets/audio`;
- налаштовує гучність;
- запускає, зупиняє та вивантажує музику.

### `modules/create_path.py`

Функція `create_path(img, folder)` будує абсолютний шлях до файлів у папці `assets`. Це дозволяє запускати гру з кореня проєкту без ручного прописування повних шляхів до ресурсів.
